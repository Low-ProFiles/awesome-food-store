## How to Start?

```
# Step 0 : 처음 시작한 경우
yarn install

# Step 1 : 실행
yarn dev

# Step 2 : JSON 서버
json-server -p 9000 db.json
```

<br/>

## Tech Stack

```
• 주요 개발 언어 및 프레임워크
```
